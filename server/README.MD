## Endpoints

### SalesPerson
- create salesperson

  Endpoint (POST): ```localhost:4000/salespersons/create```
  request body example: 
  ```
  {"firstName":"Brandy", "lastName":"Dan", "email":"bd@gmail.com", "phoneNumber":"111"}
  ```

- Get all salesperson

  Endpoint (GET): ```localhost:4000/salespersons/```
